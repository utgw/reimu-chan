# reimu-chan
A Ruby reimplementation of Marisa-chan ([utgw/marisa-chan](https://github.com/utgw/marisa-chan))

## Requirements
* Ruby 2.3.0
* Bundler

## How to use

```sh
$ gem install bundler # if you didn't install bundler
$ bundle install

# config variables
$ export TWITTER_CONSUMER_KEY="YOUR_TWITTER_CONSUMER_KEY"
...

# start!
$ ruby reimuchan.rb
```
